CMPSC132 Cache > more info

This program uses various data structures to simulate how a cache memory system works. Using object-oriented programming in Python 3.7, a hash table with the ability to use chaining to avoid collisions was used to store peices of data, identified by an ID and a header. The cache created had three levels, all of which had distinct hash indices. Methods such as put(), update(), lruEvict(), and mruEvict() were used.
