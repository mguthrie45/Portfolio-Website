My Courses

CMPEN270: Intro to Digital Design & Theory
Semester 2

CMPEN270 focused on the basics of digital design, exploring combinational logic, sequential logic, logic gates, encoders/decoders, CMOS, latches, flip flops, state machines, and SRAM cells.

Labs heavily relied on knowledge of logic gates and the ability to implement them on breadboards. The course also taught Verilog, which was a requirement for completing labs.

A noteable project for CMPEN270 was a 7-segment LED counter with Verilog.

CMPSC221: Object-Oriented Programming on the Web
Semester 3

CMPSC221 taught the basics of GUI programming with an object-oriented approach. The class also covered the material involving relational databases. The languages mainly focused on in the course was Java, but learning SQL was another requirement.

Noteworthy projects include a drawing application made with JFrames and a room reservation software using Java and Apache Derby.

See my room scheduler project on Github.
CMPSC360: Discrete Mathematics for Computer Science
Semester 3

CMPSC360 logic, inference rules, and proof techniques were major areas of study. The course moved to probability, number theory, and graph theory.

CMPEN331: Computer Organization & Design
Semester 3

CMPEN331 covered computer architecture. The course revolved around MIPS instruction set architecture and the MIPS Assembly language.

The course first studied assembly language and instruction formatting, MIPS pipeline stages, components of the CPU, hazards, and branch prediction.

The second significant portion of the course involved the study of memory hierarchy, mainly focusing on cache memory, virtual memory and their performance measures.

The final project was to simulate an entire 5-stage MIPS pipelined CPU with forwarding using Verilog.

See my 32-bit Verilog CPU project.
CMPSC465: Data Structures & Algoriths
Semester 4

Divide-and-conquer algorithms, graph algorithms, and greedy algorithms were major areas of study.

The use of correctness proofs and space/time complexity analysis when studying algorithms was strongly emphasized.

CMPSC311: Systems Programming
Semester 4

Technologies explored include the C programming language (useful functions/system calls, data types, libraries), UNIX OS, and Git.

An understanding of UNIX and its tools/environment was necessary (Ubuntu VM used for all coursework).

Explored shell programs, gcc compiler, makefiles, build sequences, memory management, caching, file systems, signal processing, socket programming, and parallel programming.

The final project was a JBOD driver interface, implementing command processing, read/write calls, and caching. Remote read/write calls for JBOD servers were also implemented using sockets.

See my JBOD driver interface with C.
CMPSC461: Programming Language Concepts
Semester 4

Topics covered include regular expressions, context-free grammar, scopes, parameter passing, functional programming, lambda calculus, types, OOP, and memory management.

The final project was an interpreter for a made up functional programming language using Racket. Types, lists, variables, functions, let blocks, and if statements were implemented.
