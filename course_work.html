My Courses

CMPSC132: Intro to Data Structures
Semester 2

CMPSC132 taught the basics of inheritance, polymorphism, abstraction, and encapsulation with Python. It also introduced students to data structures, including linked lists, stacks, queues, heaps, hash tables, and graphs. CMPSC132 also gave its students experience in sorting algorithms, searching algorithms, and big O notation.

Homework assignments primarily focused on programming and using data structures with an OOP approach.

Noteworthy projects include a calculator and equation parser using stacks and postfix notation. Programmed a simulation of cache memory using hash tables and chaining.

CMPSC221: Object-Oriented Programming on the Web
Semester 2

CMPSC221 taught the basics of GUI programming with an object-oriented approach. The class also covered the material involving relational databases. The languages mainly focused on in the course was Java, but learning SQL was another requirement when working with databases.

Noteworthy projects include a drawing application made with JFrames and a room reservation software using Java, SQL, and Apache Derby.

CMPSC360: Discrete Mathematics for Computer Science
Semester 2

CMPSC360 spent the first half of the semester discussing logic, inference rules, and proof techniques. The course then moved to probability, number theory, and graph theory.

CMPEN270: Intro to Digital Design & Theory
Semester 2

CMPEN270 focused on the basics of digital design, starting with combinational logic and moving onto sequential logic. logic gates, encoders/decoders, CMOS, latches, flip flops, state machines, and SRAM cells were all major areas of study in this course.

Labs heavily relied on knowledge of logic gates and the ability to implement them on breadboards. The course also taught Verilog, which was a requirement for completing labs.

A noteable project for CMPEN270 was a 7-segment LED counter with Verilog.

CMPEN331: Computer Organization & Design
Semester 2

CMPEN331 covered a wide range of topics. The course started with an introduction to MIPS instruction set architecture and CPU performance measures. A thorough understanding of MIPS assembly and instruction formats was necessary.

The course transitioned to MIPS pipeline stages, components of the CPU, and an in-depth look into data hazards, structural hazards, and control hazards, and branch prediction. The course touched on exceptions and interrupts.

The final significant portion of the course involved the study of memory hierarchy, cache memory and its performance measures.
