Stock Sentiment Analyzer

An application made with Tkinter, BeautifulSoup4, and Vader's sentiment analysis tool to scrape web articles relating to specific stocks. Threading was used to conduct sentiment analysis on a time interval basis while the application is running. When the user's watchlist contains a stock that has changed sentiment significantly, the user is notified through the app and by email.

Sorting Algorithm Visualizer Tool

Using HTML, CSS, and JavaScript, I made a responsive and minimalistic visualizer for the most famous sorting algorithms. Although JavaScript shows minimal support for OOP, this program heavily relied on the use of objects and classes.

Aside from learning about sorting algorithms and recursion, this project taught me the basics of JavaScript canvas animations and the use of animation arrays.

Top Down Survival Game

A canvas game made with pure JavaScript that some would describe as belonging to the "bullet hell" genre. The objective is to survive as long as possible, collecting upgrades and shooting enemy objects of varying sizes that bounce around the frame. Boss waves occur periodically to make it more difficult. Once an enemy takes too much health from you, the game is over.

Using some basic vector analysis, collision physics between the player and enemy objects were taken into account.

Animated Sudoku Solving Bot

A vizualizing tool made with Pygame. A script is used to randomly generate a solvable sudoku board that is fed into a recursive sudoku solving algorithm. The recursive nature allows for convenient backtracking to change previous cells.

Fourier Representation

Made with Matplotlib and Kivy, the user inputs a function, or pieces of functions, and the function is plotted in the form of fourier series.

The tool allows for the user to define the interval over which the fourier transform shall be executed.

The integration method, which belongs to the Function class, was programmed from scratch, implementing the trapezoidal rule.
