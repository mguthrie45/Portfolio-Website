Stock Sentiment Analyzer

An application made with Tkinter, BeautifulSoup4, and Vader's sentiment analysis tool to scrape web articles relating to specific stocks. Threading was used to conduct sentiment analysis on a time interval basis while the application is running. When the user's watchlist contains a stock that has changed sentiment significantly, the user is notified through the app. The program contains basic analysis tools and the ability to paper trade.

Search Engine from Scratch

Built with Python 3, the rudimentary search engine contains three componenets: the sqlite3 database, the main search interface, and the crawling bot. The crawling bots were designed to self-replicate to increase the discovery rate.

This project allowed me to understand how search engines work more meaningfully.

Sorting Algorithm Visualizer Tool

Using HTML, CSS, and JavaScript, I made a responsive and minimalistic visualizer for the most famous sorting algorithms. Although JavaScript shows minimal support for OOP, this program heavily relied on the use of objects and classes.

Aside from learning about sorting algorithms and recursion, this project taught me the basics of JavaScript canvas animations and the use of animation arrays.

Top Down Survival Game

A canvas game made from scratch with pure JavaScript. Some would describe this game as belonging to the "bullet hell" genre. The objective is to survive as long as possible, collecting upgrades and shooting enemy objects of varying sizes that bounce around the frame. Boss waves occur periodically to make it more difficult. Once an enemy takes too much health from you, the game is over.

Using some basic vector analysis, collision physics between the player and enemy objects were taken into account.

Neural Network Digit Classifier

Tensorflow2 was used to train a sequential neural network using MNIST digit data, which was then applied to a 28x28 PyGame drawing grid that guesses the digit drawn by the user.

Animated Sudoku Solving Bot

A vizualizing tool made with Pygame. A script is used to randomly generate a solvable sudoku board that is fed into a recursive sudoku solving algorithm. The recursive nature allows for convenient backtracking to change previous cells.
