PYLO Doctest Analyzer > more info

Often, a hurdle that I found with Python doctests is that they can output information that isn't always useful for debugging. It would be quite easier to have a doctest that is simpler and more concise. What purpose of this project was to create a program in Python 3.7 that executes a doctest on an inputted file, interprets that doctest, and returns one that is more concise. This project helped me practice with the OS and Platform pluggins for python and learning the ability to manipulate console commands with Python.
